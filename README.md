# 🧭 Career-Trek AI

> **AI-Powered Career Guidance Platform**  
> Built with **IBM Bob**, **Streamlit**, and **IBM watsonx Orchestrate**

Career-Trek AI is an intelligent AI-powered career guidance application that helps students and aspiring professionals make informed career decisions. Through an interactive conversational assistant, users can explore career opportunities, identify skill gaps, receive personalized learning roadmaps, improve resumes, prepare for interviews, and discover relevant certifications.

**Live Application:** https://career-trek-ai.streamlit.app/

## ✨ Features

- 🤖 **AI Career Assistant** – Interactive AI-powered career guidance.
- 📊 **Skill Analysis** – Identify strengths and skill gaps.
- 🛣️ **Learning Roadmap** – Personalized learning recommendations.
- 📄 **Resume & Interview Guidance** – Improve resumes and prepare for interviews.
- 🎓 **Certification Recommendations** – Explore certifications aligned with career goals.
- 💼 **Career Recommendations** – Discover career paths based on your skills and interests.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/kGopikaG48/Career-TrekAI.git
cd Career-TrekAI
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Configure Environment Variables

Create a `.env` file:

```env
IBM_API_KEY=YOUR_API_KEY
AGENT_URL=YOUR_AGENT_URL
AGENT_UID=YOUR_AGENT_UID
```

### 4️⃣ Run the Application

```bash
streamlit run app.py
```

Open your browser:

```
http://localhost:8501
```
